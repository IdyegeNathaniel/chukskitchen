# **Chuks Kitchen: Authentic Nigerian Food Delivery**
---
A  modern, full-featured food delivery application built with React, TypeScript, and Tailwind CSS. Chuks Kitchen brings the rich culinary heritage of Nigeria right to your doorstep with a seamless ordering experience.

### **Tech Stack**
---
* Framework: React 18 with Vite
* Language: TypeScript
* Styling: Tailwind CSS with custom theme
* Routing: React Router v7
* Icons: Lucide React
* Fonts: Google Fonts (Inter, Island Moments)
* Image Optimization: Vite asset handling

### **Getting Started**
---
*Prerequisites*

* Node.js 16.x or higher
* npm or yarn

### **Installation**
---
1. Clone the repository

* git clone
  ```js
  https://github.com/IdyegeNathaniel/chukskitchen.git
  ```

2. Change to project directory
```js 
cd chukskitchen
```

3. Install dependencies

```js
npm install
```
 or
```js
yarn install
```

4. Run the development server

```js
npm run dev
```
or
```js
yarn dev
```

* Open on your browser
  http://localhost:3000
  
### **Building for Production**
---
* npm run build
* npm start
